= Deployment Guide
:doctype: book
:sectnums:
:toc: left
:icons: font
:experimental:
:imagesdir: ./images


Shows how to deploy a {productnamereg}
cluster, onto bare metal or virtual machines.
Describes multiple deployment methods: by individually installing each node from ISO images, automatic deployment using AutoYAST, and also building a cluster of {xen}
or {kvm}
VMs using pre-installed virtual disk images.
